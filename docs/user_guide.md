# User Guide: Insurance CRM

> 보험설계사 고객 관리 시스템 사용 설명서

---

## 📌 시작하기

### 설치 및 실행
1. `InsuranceCRM.exe` 파일을 다운로드합니다
2. 파일을 더블클릭하여 실행합니다
3. (설치 과정 없이 바로 실행됩니다)

---

## 🎯 주요 기능 사용법

### 1. 새 고객 추가

1. **"새 고객 추가"** 버튼을 클릭합니다
2. 고객 정보를 입력합니다:
   - **이름** (필수): 고객 이름
   - **전화번호** (필수): 예) 010-1234-5678
   - **생년월일** (선택): 예) 1990-01-15
   - **주소** (선택): 고객 주소
   - **이메일** (선택): 예) example@email.com
   - **메모** (선택): 특이사항 등
3. **"저장"** 버튼을 클릭합니다
4. 목록에 즉시 추가됩니다

### 2. 고객 검색

1. 상단 **검색창**에 이름 또는 전화번호를 입력합니다
2. 입력하는 즉시 결과가 필터링됩니다
3. 검색창을 비우면 전체 목록이 표시됩니다

### 3. 고객 정보 수정

1. 목록에서 수정할 고객을 **선택**합니다
2. **"수정"** 버튼을 클릭합니다
3. 정보를 변경합니다
4. **"저장"** 버튼을 클릭합니다

### 4. 고객 삭제

1. 목록에서 삭제할 고객을 **선택**합니다
2. **"삭제"** 버튼을 클릭합니다
3. 확인 메시지에서 **"예"**를 선택합니다
4. 목록에서 즉시 제거됩니다

### 5. 데이터 백업

1. **"백업"** 버튼을 클릭합니다
2. 백업 파일을 저장할 위치를 선택합니다
3. 백업이 완료되면 확인 메시지가 표시됩니다

**권장 백업 주기**: 주 1회 또는 중요한 작업 후

### 6. 데이터 복원

1. **"복원"** 버튼을 클릭합니다
2. 복원할 백업 파일을 선택합니다
3. 확인 메시지에서 **"예"**를 선택합니다
4. 데이터가 복원됩니다

**주의**: 복원 시 현재 데이터가 백업 시점으로 되돌아갑니다

---

## ⚠️ 주의사항

### 입력 규칙
- **이름**: 반드시 입력해야 합니다
- **전화번호**: 반드시 입력해야 하며, 중복 불가
  - 올바른 형식: 010-1234-5678, 01012345678
- **이메일**: 올바른 이메일 형식만 가능
- **생년월일**: YYYY-MM-DD 형식 (예: 1990-01-15)

### 데이터 안전
- 프로그램을 종료하기 전 **백업**을 권장합니다
- `data/crm.db` 파일을 수동으로 복사해도 백업 가능합니다
- 백업 파일은 안전한 위치에 보관하세요

---

## 🐛 문제 해결

### 프로그램이 실행되지 않아요
- Windows Defender가 차단할 수 있습니다 → 허용 설정
- exe 파일을 마우스 우클릭 → "관리자 권한으로 실행"

### 고객을 추가할 수 없어요
- 전화번호가 중복되는지 확인하세요
- 이름과 전화번호를 모두 입력했는지 확인하세요

### 데이터가 사라졌어요
- 백업 파일이 있다면 "복원" 기능을 사용하세요
- `data/crm.db` 파일이 삭제되었는지 확인하세요
