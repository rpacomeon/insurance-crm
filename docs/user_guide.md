# User Guide: Insurance CRM

최소 기능으로 빠르게 사용하는 가이드입니다.

## 1) 실행
1. `scripts\run_app.bat` 실행
2. 메인 화면에서 고객 목록 확인

## 2) 고객 관리
1. `새 고객 추가` 클릭
2. 필수 입력: 이름, 전화번호
3. `저장` 클릭

수정/삭제는 목록에서 고객 선택 후 버튼 클릭.

## 3) 계약 관리
1. 고객 선택
2. 계약(Policy) 추가 또는 수정
3. 결제 방식/납부일 입력 후 저장

## 4) 검색
- 상단 검색창에 이름 또는 전화번호 입력
- 입력 즉시 필터링

## 5) 백업/복원
- `백업`: 현재 DB 파일 백업
- `복원`: 백업 파일로 현재 데이터 교체

주의: 복원하면 현재 데이터는 백업 시점 상태로 바뀝니다.

## 6) 납부 관리 필터(신용카드)
- `신용카드만 보기`: 고객의 입금방식이 `신용카드`(또는 `card`)인 고객만 표시
- `연체` 필터 정렬 규칙:
1. 연체일수 큰 순
2. 고객명 오름차순
- 운영 목적: 먼저 실효 위험이 큰 고객(장기 연체)을 상단에서 우선 처리

## 7) 실무 우선순위 권장
1. 연체
2. 오늘 카드납부
3. 이번주 준비

## 입력 규칙
- 이름: 필수
- 전화번호: 필수, 중복 불가
- 카드번호: `card_number` 기준 입력(하이픈 허용)

## 문제 해결
- 실행 불가: 보안 프로그램 차단 여부 확인
- 저장 실패: 필수값/중복값 확인
- 데이터 누락: 백업 파일로 복원
