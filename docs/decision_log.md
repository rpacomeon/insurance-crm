## 2026-02-14: 기술 스택 선정

### 결정
- **GUI**: tkinter 선택
- **DB**: SQLite 선택
- **빌드**: PyInstaller 선택

### 이유
1. **tkinter**: Python 표준 라이브러리, 의존성 최소화, 가벼움
2. **SQLite**: 서버 불필요, 파일 기반으로 백업 간편, 로컬 우선
3. **PyInstaller**: 단일 exe 파일 생성, 설치 불필요

### 대안 고려
- ~~Streamlit~~: 웹 기반이라 오프라인 사용 불편
- ~~PostgreSQL~~: 서버 설정 필요, 과도하게 복잡
- ~~cx_Freeze~~: PyInstaller가 더 안정적
